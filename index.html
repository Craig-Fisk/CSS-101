What is CSS?

CSS a.k.a. Cascading Style Sheets

  • One of the cornerstone technologies of the web
    • HTML (Content)
    • CSS (Looks & Layout)
    • Javascript (Interactivity & Data)

CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

(CSS) is standardised across web browsers according to the W3C specification. Developed in levels, CSS 1 is now obsolete,
CSS 2.1 is a recommendation, and CSS 3, noow split into smaller modules, is progressing on the standardisation track.

- MDN web docs

CSS is a stylesheet language

It is not a programming language (e.g. Javascript, Java etc) and it is not a markup language (e.g. HTML, XML, Markdown) it is a stylesheet language

The syntax allows you to selectively choose a set of elements on a HTML page and apply any number of properties to them.

div.my-class { background-color: red; }

5 types of CSS selector

CSS Selectors - Simple Selectors

Tag:

div

Class:

.block__element--modifier

ID:

#iAmUnique

CSS Selectors - Attribute Selectors

Has attribute:

[data-my-attribute]

Has attribute with value:

[data-index=5]

Has attribute and contains value:

[data-colour*="pink"]

CSS Selectors - Pseudo Class Selectors

Select an element in a given state

:hover

:first-of-type

:disabled

:last-child

:only-of-type

:focus

CSS Selectors - Pseudo Element Selectors

Select a certain part of an element

::first-letter

::before

::after

::first-line

::selection

CSS Selectors - Combinators

List (this and that):

.foo, #bar

Decendent:

#foo .bar

Direct Child:

div > p

CSS Selectors

Selectors can be combined:

#myId > div.first-class p[data-color*="red"]:hover::first-letter { }

Select the first letter of a <p> tag that is being hovered, but only if it has an attribute on it called data-color containing the string "red" in it's value, but only if it is a decendent of a <div> with the class .first-clsss on it that is a direct decendent of an element with the id "myId"

CSS Properties and Values

Within any given selector you can then set any number of properties with given values.

Some properties are only valid for certain HTML elements.

But the most common properties are generally valid on any HTML element.

Each property has a very specific list of values/keywords or specific syntax that must be used to define their values

  • margin
  • padding
  • border
  • color
  • background
  • opacity
  • font-size
  • line-height
  • transform
  • letter-spacing
  • animation
  • etc .....

CSS can be included into a HTML document in 3 ways

  • In a Separate CSS file included in the <HEAD> of the document

    <p style="color: green;"> My text </p>

How do we know which style will be applied?

2 Key factors

  • Ordering
    • If specificity of 2 selectors is the same then the one written last is applied
  • Specificity
    • Overrides ordering
